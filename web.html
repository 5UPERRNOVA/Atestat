Limbaje de programare web

HTML, CSS și JavaScript

HTML CSS JavaScript

HTML

HTML (HyperText Markup Language) este limbajul de bază pentru crearea paginilor web. El definește structura unei pagini, folosind elemente precum titluri, paragrafe, imagini și link-uri.

CSS

CSS (Cascading Style Sheets) este folosit pentru a stiliza paginile web. Cu ajutorul CSS poți schimba culorile, fonturile, dimensiunile și layout-ul unei pagini.

JavaScript

JavaScript este un limbaj de programare care face paginile web interactive. Poți crea animații, valida formulare și răspunde la acțiunile utilizatorului.
